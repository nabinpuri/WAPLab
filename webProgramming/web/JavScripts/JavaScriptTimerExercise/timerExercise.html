1. What is the difference between the following 2 statements

    setTimeout(booyah, 2000);

    setTimeout(booyah(), 2000);

    Ans: setTimeout(booyah, 2000); calls the booyah function after waiting 2000 ms but setTimeout(booyah(), 2000); calls the function immediately rather than waiting to 2000ms;
